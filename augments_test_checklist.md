# 🎲 증강 요트 다이스 (Augmented Dice) 증강 수동 테스트 체크리스트

본 문서는 증강 요트 다이스에 구현된 총 43종의 증강(Augment) 뮤테이션, 퀘스트, 주사위 변형 기능의 수동 QA 검증을 위해 제작된 상세 체크리스트입니다.

---

## 1. 족보 변형 뮤테이션 (Score Categories Mutations)

- [ ] **1. 럭키 세븐 (Lucky Sevens)**
  * **Target**: 에이스 (Aces)
  * **조건 검증**: 주사위 5개의 눈금 총합이 7, 17, 27 중 하나일 때 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 15점 정확히 기입되는지 확인.
  * **보너스 포함 검증**: 획득한 15점이 상단 보너스(63점) 계산 합계에 정상 합산되는지 확인.

- [ ] **2. 퍼펙트 스퀘어 (Perfect Squares)**
  * **Target**: 에이스 (Aces)
  * **조건 검증**: 주사위 5개의 눈금 총합이 9, 16, 25 중 하나일 때 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 12점 정확히 기입되는지 확인.
  * **보너스 포함 검증**: 획득한 12점이 상단 보너스(63점) 계산 합계에 정상 합산되는지 확인.

- [ ] **3. 안티-에이스 듀스 (Anti-Ace Deuces)**
  * **Target**: 듀스 (Deuces)
  * **조건 검증**: '2' 눈금이 3개 이상이면서 '1' 눈금이 0개일 때만 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 고정 8점 정상 등록되는지 확인.

- [ ] **4. 안티-포 트리플 (Anti-Four Threes)**
  * **Target**: 트리플 (Threes)
  * **조건 검증**: '3' 눈금이 3개 이상이면서 '4' 눈금이 0개일 때만 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 고정 12점 정상 등록되는지 확인.

- [ ] **5. 프라임 넘버즈 (Prime Numbers)**
  * **Target**: 트리플 (Threes)
  * **조건 검증**: 주사위 5개가 모두 소수(2, 3, 5)로만 구성되어 있을 때 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 12점 정상 등록되는지 확인.

- [ ] **6. 안티-식스 쿼드 (Anti-Six Fours)**
  * **Target**: 쿼드 (Fours)
  * **조건 검증**: '4' 눈금이 3개 이상이면서 '6' 눈금이 0개일 때만 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 고정 16점 정상 등록되는지 확인.

- [ ] **7. 안티-식스 펜타 (Anti-Six Fives)**
  * **Target**: 펜타 (Fives)
  * **조건 검증**: '5' 눈금이 3개 이상이면서 '6' 눈금이 0개일 때만 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 고정 20점 정상 등록되는지 확인.

- [ ] **8. 안티-파이브 헥사 (Anti-Five Sixes)**
  * **Target**: 헥사 (Sixes)
  * **조건 검증**: '6' 눈금이 3개 이상이면서 '5' 눈금이 0개일 때만 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 고정 24점 정상 등록되는지 확인.

- [ ] **9. 갬블러 (Gambler)**
  * **Target**: 초이스 (Choice)
  * **조건 검증**: 주사위 5개 눈금 총합이 24 이상일 때만 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 (총합 + 7) 점수가 정확히 기입되는지 확인 (예: 총합 25점 → 32점 기입).

- [ ] **10. 쓰리 오브 어 카인드 (Three of a Kind)**
  * **Target**: 포카인드 (Four of a Kind)
  * **조건 검증**: 같은 눈금이 3개 이상일 때 포카인드 칸에 기입 가능 상태가 되는지 확인.
  * **점수 검증**: 조건 만족 시 (총합 - 2) 점수가 정확히 기입되는지 확인 (예: 4 4 4 1 2 총합 15 → 13점 기입).

- [ ] **11. 포 바이 포 (Four by Four)**
  * **Target**: 포카인드 (Four of a Kind)
  * **달성 점수 검증**: '4'를 4개 이상 모았을 때 (총합 + 10) 점수가 기입되는지 확인 (예: 4 4 4 4 6 총합 22 → 32점).
  * **일반 달성 점수 검증**: 다른 숫자(예: 5 5 5 5 2)로 포카인드 달성 시 (총합 - 4) 점수가 기입되는지 확인.

- [ ] **12. 타이니 하우스 (Tiny House)**
  * **Target**: 풀하우스 (Full House)
  * **조건 검증**: 5와 6을 제외한 눈금(1, 2, 3, 4)으로만 풀하우스를 달성했을 때 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 28점 정확히 기입되는지 확인.

- [ ] **13. 투 페어 (Two Pair)**
  * **Target**: 풀하우스 (Full House)
  * **조건 검증**: 서로 다른 2쌍의 숫자(예: 2 2 4 4 6)가 각각 2개 이상 모였을 때 풀하우스 칸에 기입 가능한지 확인.
  * **점수 검증**: 조건 만족 시 고정 15점 정확히 기입되는지 확인.

- [ ] **14. 머리와 몸통 (Head & Tail)**
  * **Target**: 풀하우스 (Full House)
  * **조건 검증**: 연속된 눈금 3개 + 같은 눈금 2개(예: 1 2 3 + 5 5)를 구성했을 때 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 (총합 + 10) 점수가 기입되는지 확인.

- [ ] **15. 에번스 (Evens)**
  * **Target**: 스몰 스트레이트 (Small Straight)
  * **조건 검증**: 주사위 5개가 모두 짝수(2, 4, 6)일 때 스몰 스트레이트 칸에 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 20점 정확히 기입되는지 확인.

- [ ] **16. 오즈 (Odds)**
  * **Target**: 스몰 스트레이트 (Small Straight)
  * **조건 검증**: 주사위 5개가 모두 홀수(1, 3, 5)일 때 스몰 스트레이트 칸에 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 20점 정확히 기입되는지 확인.

- [ ] **17. 더블 라지 스트레이트 (Double Large Straight)**
  * **Target**: 스몰 스트레이트 (Small Straight)
  * **족보 변경 검증**: 스몰 스트레이트 족보가 라지 스트레이트(5연속) 조건으로 변경되는지 확인.
  * **보너스 기준 완화 검증**: 상단 보너스 달성 기준점수가 63점에서 60점으로 하향 완화되는지 확인.

- [ ] **18. 프라임 컬렉션 (Prime Collection)**
  * **Target**: 라지 스트레이트 (Large Straight)
  * **조건 검증**: 모든 눈금이 소수(2, 3, 5)이면서 2, 3, 5가 각각 최소 1개 이상 포함되어 있는지 확인 (예: 2 2 3 5 5).
  * **점수 검증**: 조건 만족 시 고정 35점 정확히 기입되는지 확인.

- [ ] **19. 땅콩주택 (Duplex House)**
  * **Target**: 라지 스트레이트 (Large Straight)
  * **조건 검증**: 정확히 1차이 나는 두 숫자로 이루어진 풀하우스(예: 2 2 2 3 3 또는 4 4 5 5 5)일 때 기입 가능한지 확인.
  * **점수 검증**: 조건 만족 시 고정 35점 정확히 기입되는지 확인.

- [ ] **20. 마운틴 (Mountain)**
  * **Target**: 라지 스트레이트 (Large Straight)
  * **조건 검증**: 주사위 눈금이 정확히 2, 3, 4, 5, 6 일 때만 라지 스트레이트 칸에 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 40점 정확히 기입되는지 확인.

- [ ] **21. 하이 다이스 (High Dice)**
  * **Target**: 라지 스트레이트 (Large Straight)
  * **조건 검증**: 모든 주사위가 4, 5, 6이면서 눈금 총합이 26 이상일 때 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 35점 정확히 기입되는지 확인.

- [ ] **22. 두 번째 초이스 (Whatever it takes...)**
  * **Target**: 요트 (Yacht)
  * **조건 검증**: 주사위 눈금 조합 조건 없이 언제든 요트 칸에 기입 가능한지 확인.
  * **점수 검증**: (총합 / 2) 내림 점수가 정확히 기입되는지 확인 (예: 총합 19점 → 9점 기입).

- [ ] **23. 피보나치 넘버즈 (Fibonacci Numbers)**
  * **Target**: 요트 (Yacht)
  * **조건 검증**: 주사위 눈금이 정확히 1, 1, 2, 3, 5 조합을 이룰 때만 요트 칸에 기입 가능한지 확인.
  * **점수 검증**: 조건 만족 시 고정 25점 정확히 기입되는지 확인.

- [ ] **24. 리버스 초이스 (Reverse Choice)**
  * **Target**: 요트 (Yacht)
  * **조건 검증**: 조건 없이 언제든 요트 칸에 기입 가능한지 확인.
  * **점수 검증**: (30 - 총합) 점수가 정확히 기입되는지 확인 (예: 총합 12점 → 18점 기입).

- [ ] **25. 요트 뱅크 (Yacht Bank)**
  * **Target**: 요트 (Yacht)
  * **이율 누적 검증**: 매 턴마다 뱅크 이자가 +2점씩 누적되는지 확인 (예: 2 (+4) 형태 표기).
  * **일반 기입 점수 검증**: 요트가 아닌 조합으로 기입 시 누적된 이자만큼 점수 획득 확인.
  * **요트 달성 점수 검증**: 요트 조합으로 기입 시 누적 이자의 2배 점수 획득 확인.
  * **누적 중단 검증**: 상대방이 요트를 기입하면 더 이상 이자가 동결되어 증가하지 않는지 확인.

- [ ] **26. 블랙잭 21 (Blackjack 21)**
  * **Target**: 요트 (Yacht)
  * **조건 검증**: 주사위 5개의 눈금 총합이 정확히 21일 때만 요트 칸에 기입 활성화 여부 확인.
  * **점수 검증**: 조건 만족 시 고정 21점 정확히 기입되는지 확인.

## 2. 퀘스트형 증강 (Quests)

- [ ] **27. 재빠른 스트레이트 (Fast Straight)**
  * **조건 검증**: 1\~8턴 이내에 스몰 스트레이트와 라지 스트레이트를 모두 족보에 기입하면 퀘스트 완료 처리되는지 확인.
  * **보상 검증**: 완료 시 보너스 +15점이 즉시 합산되는지 확인.

- [ ] **28. 낭비할 시간 없다 (No time to waste)**
  * **조건 검증**: 3턴 연속으로 리롤(다이스 재굴림) 없이 첫 굴림만으로 족보를 기입하면 퀘스트 완료 처리되는지 확인.
  * **보상 검증**: 완료 시 보너스 +15점이 즉시 합산되는지 확인.

- [ ] **29. 차근차근 (Step by Step)**
  * **조건 검증**: Aces부터 Sixes까지 위에서부터 순서대로 중간 건너뜀 없이 족보를 기입해야 하는지 확인.
  * **보상 검증**: Sixes 완료 즉시 상단 보너스 기준이 55점으로 강화되는지 확인.

- [ ] **30. 두 집 살림 (Two Households)**
  * **조건 검증**: '풀하우스' 족보를 기입한 후, '초이스' 칸에도 풀하우스 형태(3+2개)로 기입 시 퀘스트 완료 처리 확인.
  * **보상 검증**: 완료 시 보너스 +10점 획득 확인.

- [ ] **31. 알박기 (Holdout)**
  * **조건 검증**: 9\~12턴 사이에 풀하우스 족보를 기입하면 퀘스트 완료 처리 확인.
  * **보상 검증**: 완료 시 보너스 +7점 획득 확인.

- [ ] **32. 신중한 스트레이트 (Cautious Straight)**
  * **조건 검증**: 스몰 스트레이트를 먼저 기입한 뒤 라지 스트레이트를 기입해야 정상 완료되는지 확인 (순서 위반 시 퀘스트 실패 표시 확인).
  * **보상 검증**: 완료 시 보너스 +7점 획득 확인.

- [ ] **33. 티끌 모아 태산 (Every Little Makes a Mickle)**
  * **조건 검증**: 게임 진행 중 족보 기입 시 사용된 1 눈금 주사위의 누적 개수가 7개가 되면 퀘스트 완료 처리 확인.
  * **보상 검증**: 완료 시 보너스 +15점 획득 확인.

## 3. 주사위 변형 및 패시브 증강 (Dice & Passive Mutations)

- [ ] **34. 묵직한 주사위 (Weighted Dice)**
  * **주사위 변형 검증**: 기본 주사위 1개가 묵직한 주사위(눈금 구성: 4, 4, 5, 5, 6, 6)로 변경되어 굴려지는지 확인.

- [ ] **35. 럭키 펀치 (Lucky Punch)**
  * **패시브 검증**: 매 턴 첫 번째 굴림(리롤 0회)만으로 하단 구역 족보 기입 시 획득 점수에 +5점 보너스가 합산되는지 확인.

- [ ] **36. 추진력 (Momentum)**
  * **발동 조건 검증**: 족보에 최초로 0점(스크래치)을 기록했을 때 추진력 상태가 'Active'로 발동하는지 확인.
  * **효과 검증**: 다음 턴에 득점 기입 시 해당 획득 점수(보너스 포함)가 1.5배로 증가되어 기입되는지 확인 (1회 적용 후 해제).

- [ ] **37. 황금 주사위 (Golden Die)**
  * **주사위 변형 검증**: 기본 주사위 1개가 황금 주사위로 변경되는지 확인.
  * **효과 검증**: 족보 기입 시 황금 주사위의 눈금이 1, 2, 3 중 하나라면 +3점 추가 보너스가 부여되는지 확인.

- [ ] **38. 아직 안 끝났어 (It's Not Over Yet)**
  * **즉시 효과 검증**: 획득 즉시 기존 기입된 족보 중 가장 점수가 낮은 족보 1개가 빈칸으로 되돌아가고 1턴이 추가 획득되는지 확인.
  * **재기입 효과 검증**: 초기화된 해당 족보를 다시 기입할 때 +3점 추가 보너스가 지급되는지 확인.

- [ ] **39. 8면 주사위 (8-Sided Dice)**
  * **주사위 변형 검증**: 기본 주사위 1개가 8면 주사위(눈금 구성: 1, 2, 3, 4, 4, 5, 5, 6)로 변경되는지 확인.

- [ ] **40. 이상한 주사위 (Strange Die)**
  * **주사위 변형 검증**: 기본 주사위 1개가 조커 이상한 주사위(면: +2, +1, +1, 0, -1, 파괴)로 변경되는지 확인.
  * **보너스/파괴 검증**: 눈금 표시만큼 점수 가감 조율 확인 및 '파괴' 면이 나온 상태로 기입 시 해당 주사위가 즉시 소멸하는지 확인.

- [ ] **41. 프로모션 주사위 (Promotion Die)**
  * **주사위 변형 검증**: 주사위 1개가 프로모션 주사위로 바뀌고 매 턴 눈금이 +1씩 상승(최대 6)하는지 확인.
  * **소모 검증**: 6의 눈 상태에서 족보 기입 시 일반 주사위로 되돌아오는 지 확인.

- [ ] **42. 커플 주사위 (Couple Dice)**
  * **주사위 변형 검증**: 커플 주사위로 지정된 두 주사위의 눈금이 일치한 상태로 족보 기입 시 +3점 추가 보너스(상단 보너스 합계에 포함) 지급 확인.

- [ ] **43. 세븐스 다이스 (Sevens Dice)**
  * **주사위 변형 검증**: 주사위 2개가 세븐스 다이스(눈금 구성: 2, 3, 4, 5, 6, 7)로 변경되는지 확인.
  * **7의 눈 연동 검증**: 7의 눈으로 족보 조건이 정상 확장 판정되는지 확인.
