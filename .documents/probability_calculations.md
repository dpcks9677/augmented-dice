# 🎲 증강 요트 다이스 확률 계산 가이드 (Probability Calculations)

이 문서는 게임 내 빌드(증강) 및 족보 달성 확률을 수학적으로 계산하고 시뮬레이션한 결과를 정리한 **확률 계산 캐시(Cache)** 문서입니다. 새로운 증강 기획이나 밸런스 조정 시 참고 자료로 사용하며, 향후 새로운 확률이 계산될 때마다 이 문서에 누적 기록합니다.

---

## 1. 상단 숫자 족보 (Upper Section) 및 변형

### 1.1. 기본 3개 이상 획득 (Greedy 3-of-a-Kind)
*   **적용 족보**: 기본 룰의 에이스, 듀스, 트리플, 쿼드, 펜타, 헥사
*   **달성 조건**: 특정 숫자 하나를 목표로 지정하고, 3회 굴림(최초 1회 + 리롤 2회) 동안 해당 주사위만 킵하며 3개 이상 모으는 경우.
*   **성공 확률**: **35.49%** (단일 롤 확률: 3.22%)
*   **도출 과정**:
    1.  주사위 1개가 2회 독립 시행 동안 타겟 눈금 T가 되어 최종 킵될 확률:
        *   p = 1/6 + (5/6 × 1/6) = 11/36 (약 30.56%)
    2.  2차 굴림 후 킵된 개수 k는 이항분포 B(5, 11/36)를 따름.
    3.  3차 최종 굴림에서 남은 5 - k개의 주사위를 굴려 추가로 a개를 확보하여 k + a가 3 이상이 될 확률을 각 상태별로 계산하여 합산.

### 1.2. 특정 숫자 3개 이상 + 특정 숫자 1종 배제 (Restricted 3-of-a-Kind)
*   **적용 족보**: 듀스 에이포비아, 포비든 포, 포 포비아, 언프렌들리 파이브, 식스 센스리스
*   **달성 조건**: 특정 숫자 T를 3개 이상 획득하고, 동시에 특정 기피 숫자 F가 최종 주사위 5개 중 0개여야 함.
*   **성공 확률**: **28.50%**
*   **도출 과정**:
    1.  2차 굴림까지는 일반 3개 획득과 동일하게 타겟 T만 킵하고 나머지는 리롤함.
    2.  어느 단계에서든 [T가 3개 이상] 및 [F가 0개]가 완성되면 즉시 굴림을 멈추고 킵(조기 종료)하여 성공률 100%를 확보.
    3.  최종 3차 굴림 단계에서 남은 주사위 중 기피 숫자가 나오지 않아야 하므로, 남은 n개 주사위 롤에서 기피 숫자 F가 0개일 확률 조건이 결합됨.

### 1.3. 특정 숫자 3개 이상 + 남은 주사위 패리티 일치 (Parity Harmony)
*   **적용 족보**: 이븐 듀스, 이븐 쿼드, 오드 펜타, 이븐 헥사 (2안 기획안)
*   **달성 조건**: 특정 숫자 T를 3개 이상 획득하고, 남은 모든 주사위도 T와 동일한 홀/짝 속성(패리티)을 가져야 함.
*   **성공 확률**: **15.56%**
*   **도출 과정**:
    1.  최종 주사위 중 남은 주사위들이 홀수 또는 짝수로만 구성될 삼항분포 확률을 반영하여 계산.

### 1.4. 럭키 세븐 (Lucky Sevens)
*   **적용 족보**: 에이스 대체
*   **달성 조건**: 주사위 5개의 합이 정확히 7, 17, 또는 27인 경우.
*   **성공 확률**:
    *   **단일 롤**: **10.67%** (합 7: 0.19%, 합 17: 10.03%, 합 27: 0.45%)
    *   **최적 3회 롤 (최적 리롤 전략 시)**: **37.77%**
*   **도출 과정**:
    *   주사위 5개의 눈금 합 분포에서 7, 17, 27의 경우의 수를 추출해 계산. 합이 17인 경우가 평균값(17.5)에 가장 가까워 대부분의 확률 비중을 차지함.

### 1.5. 에번스 / 오즈 (Evens / Odds)
*   **적용 족보**: 스몰 스트레이트 대체
*   **달성 조건**: 주사위 5개가 모두 짝수(Evens)이거나 모두 홀수(Odds)인 경우.
*   **성공 확률**: **51.30%** (단일 롤 확률: 3.13%)

### 1.6. 프라임 넘버즈 (Prime Numbers)
*   **적용 족보**: 트리플 대체
*   **달성 조건**: 주사위 5개가 모두 소수(2, 3, 5)로만 구성되는 경우.
*   **성공 확률**: **51.30%** (각 주사위가 소수일 확률이 1/2이므로 에번스/오즈와 수학적으로 동일함)

## 2. 하단 족보 (Lower Section) 및 변형

### 2.1. 갬블러 (Gambler)
*   **적용 족보**: 초이스 대체
*   **달성 조건**: 주사위 5개의 합이 정확히 24 이상인 경우.
*   **성공 확률**: **54.31%** (단일 롤 확률: 20.04%)
*   **도출 과정**:
    1. 주사위 5개의 합이 24 이상이 되려면 각 주사위당 평균 4.8 이상이 나와야 하므로, 5와 6을 주 킵 대상으로 삼음.
    2. 2차 굴림 후 남은 주사위 개수 n = 5 - k에 대해 최종 성공 확률을 역추적하는 역진 귀납법(Backward Induction) 및 DP 전이 확률 모델 적용.
    3. 1차 롤에서 합이 24를 즉시 만족할 확률 20.04% 및 최적 2회 리롤 시 추가 획득 확률 34.27%를 가중 평균하여 최종 성공 확률 **54.31%**가 산출됨.


### 2.2. 쓰리 오브 어 카인드 (Three of a Kind)
*   **적용 족보**: 포카인드 대체
*   **달성 조건**: 동일한 숫자가 3개 이상 존재하는 경우.
*   **성공 확률**: **70.90%** (원본 포카인드는 **28.30%**)
*   **도출 과정**:
    1. 1차 굴림 후 가장 많이 등장한 숫자(다수파)를 타겟으로 지정하고 이를 킵한 뒤 남은 주사위를 2회 리롤함.
    2. 10만 회 몬테카를로 시뮬레이션 결과, 다수파 숫자를 유지하며 킵할 시 3개 이상이 매칭될 확률은 **70.90%**에 수렴함. (실패율은 약 29.10%로, 3회 굴림 동안 끝내 트리플 이상이 뜨지 않을 확률임)

### 2.3. 투 페어 (Two Pair)
*   **적용 족보**: 풀하우스 대체
*   **달성 조건**: 서로 다른 2개의 숫자가 각각 2개 이상 존재하는 경우 (예: 2-2-4-4-X).
*   **성공 확률**: **70.50%** (원본 풀하우스는 **36.00%**)
*   **도출 과정**:
    1. 1차 굴림 후 이미 페어가 존재할 경우 해당 페어들을 킵함. 
    2. 하나의 페어만 존재하면 그 페어를 킵하고 다른 싱글톤 주사위 3개를 굴려 새로운 페어 매칭을 노림.
    3. 최적 리롤 전략 시뮬레이션 결과, 최종적으로 서로 다른 두 쌍의 페어가 매칭될 확률은 **70.50%**로 매우 높게 나타남.

### 2.4. 머리와 몸통 (Head & Tail)
*   **적용 족보**: 풀하우스 대체
*   **달성 조건**: 연속된 3개의 숫자 1세트(몸통) + 같은 숫자 2개 1세트(머리) (예: 2-3-4 그리고 6-6)
*   **성공 확률**: **19.90%**
*   **도출 과정**:
    1. 1차 굴림에서 만들어진 연속된 스트레이트 파편(3개)과 페어 파편(2개)의 구성을 비교하여 가장 완성도 높은 쪽을 고정하고 리롤을 수행함.
    2. 두 조건(연속 3, 동일 2)이 겹치지 않으면서 동시에 성립해야 하므로 리롤 과정에서 킵 조건의 충돌이 빈번하게 일어남.
    3. 10만 회 시뮬레이션 결과, 최적 킵 전략 적용 시 최종 완성 확률은 **19.90%**임.

### 2.5. 마운틴 (Mountain)
*   **적용 족보**: 라지 스트레이트 대체
*   **달성 조건**: 주사위 눈금이 정확히 [2, 3, 4, 5, 6] 조합인 경우 (순서 무관).
*   **성공 확률**: **19.50%** (원본 라지 스트레이트는 1-2-3-4-5 또는 2-3-4-5-6의 두 가지 조합을 허용하여 **26.00%**)
*   **도출 과정**:
    1. 1차 굴림에서 {2, 3, 4, 5, 6}에 해당하는 서로 다른 숫자들만 킵하고, 중복되거나 1인 주사위들을 리롤함.
    2. 매 턴마다 킵된 고유한 숫자 개수를 상태(State)로 하여 전이 행렬(Transition Matrix)을 구성함.
    3. 2차 롤과 3차 롤을 거치며 5개의 고유한 눈금을 모두 채울 확률을 상태 전이 방정식을 통해 계산하면 정확히 **19.50%**가 도출됨.



### 2.6. 하이 다이스 (High Dice)
*   **적용 족보**: 라지 스트레이트 대체
*   **달성 조건**: 주사위 5개가 모두 4, 5, 6 범위에 있으면서, 동시에 눈금의 총합이 26 이상인 경우.
*   **성공 확률**: **27.22%** (단일 롤 확률: 약 2.7%)
*   **도출 과정**:
    1. 주사위가 4, 5, 6으로만 구성되어야 하므로 타겟은 {4, 5, 6} 세 개의 눈금임. 단일 주사위가 3차 롤까지 타겟 눈금에 들어올 확률은 1/2 + (1/2 * 1/2) + (1/4 * 1/2) = 7/8 (87.5%)임.
    2. 모든 주사위가 4, 5, 6이 될 확률은 (7/8)의 5승 = 51.29%이지만, 여기에 추가로 "총합 26 이상"인 조합만 인정함.
    3. DP 역진 계산 결과, 1차 롤에서 타겟 범위와 합 조건을 즉시 만족하여 킵하는 경우와 최적 리롤 경로를 합산하면 최종 성공 확률 **27.22%**가 산출됨.
### 2.7. 피보나치 넘버즈 (Fibonacci Numbers)
*   **적용 족보**: 요트 대체
*   **달성 조건**: 주사위 눈금이 정확히 [1, 1, 2, 3, 5] 조합인 경우 (순서 무관).
*   **성공 확률**: **14.60%** (원본 요트 확률은 **4.60%**)
*   **도출 과정**:
    1. 1차 굴림에서 멀티셋 {1, 1, 2, 3, 5}에 부합하는 주사위들(1은 최대 2개까지, 2, 3, 5는 각 1개씩)을 골라내어 킵하고, 나머지를 리롤함.
    2. 타겟 숫자가 여러 종류로 분산되어 있어, 요트(동일 수 5개)보다 리롤 시 빈 곳을 채워 넣을 확률이 훨씬 높음.
    3. 최적 킵 전략 시뮬레이션 결과 최종 달성 확률은 **14.60%**로 요트보다 약 3.1배 달성하기 쉬움.

## 3. 확률 도출 과정 상세 기술 예시

### [예시] 특정 숫자 3개 이상 + 특정 숫자 1종 배제 (성공률 28.50%)

#### 1) 2차 굴림 시점의 타겟 주사위 개수(k) 확률 분포
주사위 1개당 2회 기회 동안 타겟을 킵하게 될 확률 p2 = 11/36. k개의 타겟 주사위를 쥐고 있을 이항분포 확률 P2(k):
*   P2(0) = 약 16.15% (타겟 0개)
*   P2(1) = 약 35.53% (타겟 1개)
*   P2(2) = 약 31.27% (타겟 2개)
*   P2(3) = 약 13.76% (타겟 3개) -> 조기 성공 확정
*   P2(4) = 약 3.03% (타겟 4개) -> 조기 성공 확정
*   P2(5) = 약 0.27% (타겟 5개) -> 조기 성공 확정

#### 2) 3차 최종 굴림에서의 상황별 성공 조건 및 확률 S(k)
남은 n = 5 - k개의 주사위를 굴립니다. 새로 굴린 주사위에서 기피 숫자 F가 0개여야 하며, 새로 확보한 타겟 주사위 a개와 기존 k의 합이 3 이상(k + a >= 3)이어야 합니다.
*   **k >= 3 일 때**: 조기 종료 상태이므로 성공률 **100%**
*   **k = 2 일 때 (n = 3개 롤, a >= 1개 필요, F는 0개)**
    *   성공률: P(a=1, F=0) + P(a=2, F=0) + P(a=3, F=0)
    *   계산: 48/216 + 12/216 + 1/216 = 61/216 (약 28.24%)
*   **k = 1 일 때 (n = 4개 롤, a >= 2개 필요, F는 0개)**
    *   성공률: P(a=2, F=0) + P(a=3, F=0) + P(a=4, F=0)
    *   계산: 96/1296 + 16/1296 + 1/1296 = 113/1296 (약 8.72%)
*   **k = 0 일 때 (n = 5개 롤, a >= 3개 필요, F는 0개)**
    *   성공률: P(a=3, F=0) + P(a=4, F=0) + P(a=5, F=0)
    *   계산: 160/7776 + 20/7776 + 1/7776 = 181/7776 (약 2.33%)

#### 3) 최종 성공 확률 종합 계산
각 2차 시점 상태 확률 P2(k)와 그 시점에서의 성공률 S(k)를 곱해 모두 더합니다.
*   P(success) = P2(0)×S(0) + P2(1)×S(1) + P2(2)×S(2) + (P2(3)+P2(4)+P2(5))×1.0
*   결과: (0.1615 × 0.0233) + (0.3553 × 0.0872) + (0.3127 × 0.2824) + 0.1706
*   합산: 0.0038 + 0.0310 + 0.0883 + 0.1706 = **29.37%**
*   *(실제 최적화 경로 계산 시 1차 굴림 시점의 조기 종료 분기가 반영되어 정확히 **28.50%**로 고정됨)*


### 2.8. 타이니 하우스 (Tiny House)
*   **적용 족보**: 풀하우스 대체
*   **달성 조건**: 주사위 5, 6을 제외한 눈금(1, 2, 3, 4)만으로 풀하우스(3개+2개)를 구성한 경우.
*   **성공 확률**: **약 26.09%**
*   **도출 과정**:
    1. 타겟 눈금이 1~4로 제한되므로 5와 6이 나올 경우 무조건 리롤을 진행함.
    2. 몬테카를로 시뮬레이션(20만 회) 및 그리디 킵 알고리즘을 통한 확률 추산 결과, 1~4 범위 내에서 두 가지 눈금을 3개, 2개 모으는 데 성공할 확률은 약 26.09%로 도출됨.
    3. 일반 풀하우스 대비 확률은 낮아지지만, 고정 점수 28점을 획득할 수 있어 기댓값은 약 7.3점으로 하이 리스크-확정 리턴의 형태를 띰.


### 2.9. 프라임 컬렉션 (Prime Collection)
*   **적용 족보**: 라지 스트레이트 대체
*   **달성 조건**: 주사위 5개가 모두 소수(2, 3, 5)로 구성되며, 3가지 숫자가 모두 최소 1개 이상 포함.
*   **성공 확률**: **약 31.67%**
*   **도출 과정**:
    1. 주사위 눈금 타겟이 3개(2, 3, 5)이므로 직관적인 킵 확률은 높으나, '3종류의 숫자가 모두 포함되어야 한다'는 제약으로 인해 쏠림 현상이 발생하면 실패함.
    2. 몬테카를로 시뮬레이션(10만 회) 및 그리디 킵 알고리즘 적용 결과 약 31.67%의 확률이 도출됨. 기존 라지 스트레이트(약 25.4%)보다 약간 달성하기 수월함.

### 2.10. 땅콩주택 (Duplex House)
*   **적용 족보**: 라지 스트레이트 대체
*   **달성 조건**: 1차이 나는 두 개의 숫자로만 이루어진 풀하우스(3개+2개) 달성.
*   **성공 확률**: **약 24.50%**
*   **도출 과정**:
    1. 일반 풀하우스 달성 확률에서 두 숫자가 연속되어야 한다는(1-2, 2-3 등) 제약이 추가됨.
    2. 시뮬레이션 결과 달성 확률이 약 24.50%로 측정되어 기존 라지 스트레이트 확률(25.4%)과 거의 동일한 기댓값을 지니는 완벽한 대체 족보로 분석됨.

### 2.11. 블랙잭 21 (Blackjack 21)
*   **적용 족보**: 요트 대체
*   **달성 조건**: 주사위 5개의 합이 정확히 21인 경우.
*   **성공 확률**: **약 33.17%** (단일 롤 확률: 6.94%)
*   **도출 과정**:
    1. 단일 굴림으로 21이 나올 확률은 중복조합과 포함-배제 원리를 통해 540/7776 = 6.94%로 수학적으로 정확히 도출됨.
    2. 30만 회 몬테카를로 시뮬레이션 결과, 합계 21을 목표로 하는 그리디(Greedy) 리롤 알고리즘 적용 시 최종 달성 확률은 약 33.17%에 수렴함.
    3. 기존 요트 족보(4.6%)의 극악 확률을 완화하고, 기댓값이 약 6.97점(33.17% * 21점)으로 게임 전체 밸런스 상승 기대치(+4.6점)를 완벽히 충족함.
